Visualping monitors an HTTP snapshot.

If your competitor runs on React or Next.js (most of them do), the snapshot is an empty div.

It'll happily tell you "no change detected" forever. Because the shell HTML never changed. The content is all JavaScript-rendered.

Headless browser is the only fix. Most tools don't do this.
