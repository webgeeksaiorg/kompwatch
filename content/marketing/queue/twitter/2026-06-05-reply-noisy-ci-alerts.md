The noise problem is structural, not fixable with more filters.

Raw HTML diffs catch cookie consent banners. CSS class name changes. A/B test variants. Ad script updates.

The only fix is AI that understands what "changed" means semantically. Not pattern matching — actual comprehension of whether the business meaning shifted.

Most tools haven't solved this yet. Some are trying.
