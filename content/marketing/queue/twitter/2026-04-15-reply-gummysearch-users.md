GummySearch did the Reddit side of this well — keyword alerts, intent monitoring, surfacing "alternatives to X" threads.

That part of the problem (social mentions, Reddit discussions) is genuinely hard to replace since they lost API access.

For the other half — tracking what changes on competitor websites directly — that's a solvable problem. Playwright + specific CSS selectors + LLM to translate the diff. Works whether the page is static or JS-heavy.

Different tools for different problems. Don't let one gap make you think the whole workflow is dead.
