Shipped an MCP server for KompWatch.

Means your AI tools — Claude, Cursor, whatever you're using — can now query your competitor change feed directly. No copy-paste. No switching tabs. Ask "what changed on Competitor X this week" and get an answer with citations.

4 tools: list-competitors, get-competitor, search-changes, get-digest.

I built this mostly because I was tired of exporting CSVs to paste into Claude. The irony of using AI to monitor competitors but not connecting the two.

Early. Rough around the edges. Works.
