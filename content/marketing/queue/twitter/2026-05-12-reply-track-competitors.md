Google Alerts for name mentions + a page monitor for pricing/changelog pages. The key is CSS selectors targeting the content section — not the full page.

Full-page monitoring on a React site is 95% noise. Session IDs and tracking pixels look like "changes" to a dumb diff.

Been running this on 8 competitors for 18 months. ~2 real signal alerts per week total.
