You can build the core of a competitor monitoring tool in a weekend.

Playwright to render JS-heavy pages. CSS selector targeting the section you care about (not `body`). LLM API call to turn the HTML diff into plain English. ~200 lines of Python.

Reasons teams don't:
- Playwright system deps in Docker are annoying
- Login-gated pricing pages
- Rate limiting / bot detection
- That one competitor who puts their features in a PDF
- It breaks, then nobody fixes it, then you're back to the manual tab-switching Monday routine

Build vs. buy at $28K/year is a real debate. At $49/month it's a different calculation.
