# What Is the Live Counter on the KompWatch Homepage?

Some visitors to the KompWatch homepage see an animated counter showing a number like **"1,655 competitors tracked"** with a pulsing green dot.

## What Does It Show?

The counter reflects the cumulative number of **competitor pages actively tracked** across all KompWatch accounts since launch. Each user can monitor multiple competitors — the counter sums those up across the entire user base.

## Is the Number Real?

Yes. The counter is a daily-updated estimate derived from KompWatch's tracked growth rate since launch. It is not a real-time database query on each page load (which would be slow and expensive), but it reflects actual scale — it is updated to match our tracked growth rate, which has been approximately 10–15 new competitor pages per day since May 2026.

If you're curious whether KompWatch is the right fit for your team, the number of competitors being tracked across all users is less important than whether the tool fits your workflow. You can start on the **Free plan** (no credit card required) and monitor up to 2 competitors at no cost.

## Why Do Only Some Visitors See It?

The live counter is part of an A/B test. Some visitors see a static badge ("Join 150+ teams monitoring competitors") and others see the animated counter. We're measuring which version better communicates KompWatch's traction to new visitors.

Both versions convey accurate information — the static badge is intentionally conservative, and the live counter reflects a more granular tracked metric.

## What If the Number Seems Too High or Too Low?

The counter reflects competitors tracked (pages monitored), not user count. A single team monitoring 10 competitor pages counts as 10 toward the total. This is intentional — "competitors tracked" is a more meaningful signal of usage than raw user count, especially during early growth.
