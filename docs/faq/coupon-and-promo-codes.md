# Do You Have Coupon or Promo Codes?

KompWatch occasionally offers promotional pricing through partner programs, launch events, and limited-time campaigns. Here's how discounts work and where to find them.

## Are There Any Active Promo Codes?

Promo codes are issued periodically — not permanently available. If you found KompWatch through a partner, community, or referral link, you may have been given a code. Apply it during checkout.

**Where to find active offers:**
- Check any email we've sent you — promo codes are distributed to our newsletter list first
- Look at our [Pricing page](/pricing) — active site-wide promotions are shown there
- Follow [@KompWatch](https://twitter.com/kompwatch) — we announce limited offers on social
- Check our ProductHunt or AppSumo listings if you came from there

If you don't have a code and are evaluating KompWatch for a team, email [support@kompwatch.com](mailto:support@kompwatch.com) — we occasionally offer evaluation discounts for teams with 5+ seats.

## How Do I Apply a Promo Code?

Promo codes are entered at checkout:

1. Go to **Settings → Upgrade** (or click a pricing CTA from the landing page)
2. You'll be redirected to a Stripe Checkout page
3. Look for the **"Add promotion code"** field — click it to expand
4. Enter your code and click **Apply**
5. The discount will appear in the order summary before you confirm

Codes cannot be applied retroactively to completed charges. If you forgot to apply a code and were just charged, email [support@kompwatch.com](mailto:support@kompwatch.com) within 7 days and we'll issue a manual credit if the code was valid.

## Can I Stack Multiple Codes?

No — only one promo code can be applied per subscription. If you have more than one code, use the one with the higher value.

## Do Codes Apply to Annual Plans?

Yes, unless the promotion is explicitly labeled "monthly only." Annual plan discounts stack with promo codes — if a code gives 20% off and you choose annual billing (also 20% off), the code may apply on top of the annual rate depending on how it's configured. The Stripe checkout preview will show your exact final price before you confirm.

## Do Codes Expire?

Yes. Most codes have an expiry date or a maximum number of uses. If your code shows as invalid:
- It may have expired
- It may have reached its redemption limit
- It may only apply to a specific plan tier

Email [support@kompwatch.com](mailto:support@kompwatch.com) with the code and we'll confirm its status.

## I'm a Student / Non-Profit / Open-Source Project

We don't have a formal program yet, but we handle these case by case. Email [support@kompwatch.com](mailto:support@kompwatch.com) with a brief description of your project and we'll see what we can do.

## Annual Billing Is Already the Best Discount

If you're price-sensitive, note that **annual billing saves 20%** compared to paying month-to-month — no code required. See [Annual Billing](./annual-billing.md) for details.
